   2 * # hsGraphD3

   3 * 

   4 * Helpful Scripts Graph convenience wrapper for D3js. 

   5 * [`[Github page]`](https://github.com/HelpfulScripts/hsGraphd3)

   6 * [`[Coverage Info]`](./data/src/hsGraph/coverage/)

   7 * ___

   8 * `hsGraphD3` provides convienent programmatic shortcuts to plotting data with [`D3js`](https://d3js.org/).

   9 * 

  10 * ## Usage

  11 * 1. create a {@link Graph `Graph`} object with a root DOM element to attach to: `graph = new hsGraphD3.Graph(root);'

  12 * 2. add desired series configuartions, specifying the type of series, as well as the column names to use for the x- and y-axes, 

  13 * and other variables, depending on the series type.

  14 * 3. render the graph for a given data set: `graph.render(data);`

  15 * 

  16 * ## Data

  17 * Data is provided either as a [`Data`](https://helpfulscripts.github.io/hsDatab/#!/api/hsDatab/0) object, or in the form 

  18 * ```

  19 * {

  20 * name?:  string;

  21 * colNames:  string[];   

  22 * rows:   [][];

  23 * }

  24 * ```

  25 *
