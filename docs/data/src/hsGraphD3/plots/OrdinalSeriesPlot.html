   2 * # OrdinalSeriesPlot

   3 * 

   4 * Abstract base class for all ordinal series plot types.

   5 * To create a series plot, add the desired plot type to the graph:

   6 * ```

   7 * graph.addSeries(, {: , ...});

   8 * ``` 

   9 * - `` is one of the registered types: 

  10 *     -   {@link plots.Bar `bar`} a horizontal bar chart

  11 *     -   {@link plots.Column `column`} a vertical column chart

  12 * - `` is one of the semantic dimensions defined for the plot. `NumericSeriesPlots` define the 

  13 *   following dimensions:

  14 *     - `x`: required; the value to plot along the x-axis

  15 *     - `y`: required; the value to plot along the y-axis

  16 * - `` is the value to use for the {@link Series.SeriesDimensions semantic dimension}. Values may be specified

  17 *     - by `string` to specify the column name of the data set to use

  18 *     - by `number` to specify a constant value to use.  

  19 */

  40/**

  41 * Returns a function to access the numeric value in a data row.

  42 * The numeric value is specified by the `ValueDef` `v`. 

  43 * - If `v` is a number, applies the `scale` to `v`, and returns the result.

  44 * - If `v` is contained in `colNames`, uses the position of `v` in `colNames` to index the 

  45 * supplied `row`, applies the `scale`, and returns the result.

  46 * - Otherwise, if `v` ends with 'u', interprets `v` to be Viewport Units and 

  47 * returns `v` without aplying `scale`. This allows for absolute positioning inside the 

  48 * viewport window.

  49 * @param v the `ValueDef` specifying the value

  50 * @param colNames a list of names for the coluymns in the `DataSet`

  51 * @param scale a numeric scale to apply 

  52 * @param def defaults to `0`; the default to return if `v` is `undefined`.

  53 */

  54export function accessor(v:ValueDef, colNames:string[], scale:Scale, def=0):(row?:number[], i?:number) => number {

  55    const index = colNames.indexOf(''+v);

  56    const fn = typeof(v)==='function';

  57    // const unit = (''+v).slice(-1);

  58    // if (index < 0 && unit === 'u') {}

  59    return v===undefined? 

  60        () => def :      // always return default

  61        (row?:number[], i?:number) => scale(fn? (v)(i) : row[index]);

  62}
