  17 * `hsGraphD3` provides convienent programmatic shortcuts to plotting data with [`D3js`](https://d3js.org/).

  18 * 

  19 * ## Usage

  20 * 1. create a {@link Graph `Graph`} object with a root DOM element to attach to: 

  21 * `graph = new hsGraphD3.GraphCartesian(root);`

  22 * 2. add desired series configurations, specifying the type of series, as well as the data to use for the x- and y-axes, 

  23 * and other variables, depending on the series type:

  24 * `graph.addSeries('area', {x:'time', y:'volume', r:5});`

  25 * 3. apply any desired formatting changes to the default:

  26 * `graph.defaults.axes.color = '#88f';`

  27 * 4. render the graph for a given data set: 

  28 * `graph.render(data);`

  29 * 

  30 * To dynamically update the graph to new data, call the {@link Graph.RenderChain `update`} method

  31 * 

  32 * ## Data

  33 * Data is provided either in table form

  34 * ```

  35 * [

  36 *    colNames[],

  37 *    [],

  38 *    ...

  39 * ]

  40 * ```

  41 * or in object-literal form 

  42 * ```

  43 * {

  44 *    name?:  string;

  45 *    colNames:  string[];   

  46 *    rows:   [][];

  47 * }

  48 * ```

  59 *             ['5/1/17',  0.6,  -0.2,    0.1], 

  60 *             ['7/1/18',  0.8,   0.3,    0.5],

  62 * };

  63 * 

  64 * // create the graph and define the series to plot:

  65 * const graph = new hsGraphD3.GraphCartesian(root);

  66 * graph.addSeries('bubble', {x:'time', y:'volume', r:'costs'});

  67 * 

  68 * // adjust some settings:

  71 *      min = 10;                       // min marker size
